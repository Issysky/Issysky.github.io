    二叉搜索树第k大节点

    郑奥威 2022/11/13 树

    # 题目

    给定一棵二叉搜索树，请找出其中第 k 大的节点的值。

    # 解法

    关键点是二叉搜索树的==中序遍历是递增的==，所以我只需要中序遍历第K大的数据。

    class Solution {
    int res, k;
    public int kthLargest(TreeNode root, int k) {
        this.k = k;
        dfs(root);
        return res;
    }
    void dfs(TreeNode root) {
        if(root == null) return;
        dfs(root.right);
        if(k == 0) return;
        if(--k == 0) res = root.val;
        dfs(root.left);
    }
}

    中序遍历的这个dfs是重点，多看看 面试题54. 二叉搜索树的第 k 大节点（中序遍历 + 提前返回，清晰图解） - 二叉搜索树的第k大节点 - 力扣（LeetCode） (opens new window)
